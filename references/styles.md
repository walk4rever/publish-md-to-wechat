# WeChat Publisher Style Guide

Available styles for Markdown to WeChat conversion. Each style uses professional inline CSS compatible with WeChat's official editor.

| Style | Vibe | Best For | Typical Colors |
| :--- | :--- | :--- | :--- |
| `swiss` | Clean, high-contrast | Technical guides, official reports | White BG, Red Accent, Black Text |
| `terminal` | Hacker aesthetic | Dev tools, coding tips, CLI updates | Dark BG, Green Accent, Monospace font |
| `bold` | High impact cards | Product launches, announcements | Dark BG, Orange Accent, Thick Borders |
| `botanical` | Elegant, premium | Artistic pieces, luxury brands, poetry | Black BG, Gold Accent, Serif font |
| `notebook` | Tactile, cream paper | Study notes, diaries, personal thoughts | Cream BG, Mint Accent, Serif font |
| `cyber` | Futuristic, neon glow | AI, web3, futuristic tech | Navy BG, Cyan Accent, Modern font |
| `voltage` | Energetic, creative | Creative pitches, music, events | Blue BG, Neon Yellow Accent |
| `geometry` | Soft, approachable | Friendly tutorials, lifestyle | Soft Pastels, Rounded corners |
| `editorial` | Personality-driven | Opinions, blogs, personal brands | Cream BG, Serif font, minimalist |
| `ink` | Literary, traditional | Storytelling, deep dives, history | Cream BG, Crimson Accent |

## Recommendations
- **Technical/Code heavy**: Use `terminal` or `swiss`.
- **Product Launch**: Use `bold` or `voltage`.
- **Personal/Blog**: Use `editorial` or `ink`.
- **Creative/Art**: Use `botanical` or `geometry`.
